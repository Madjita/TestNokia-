Проверить работоспособность программ возможно на
http://cpp.sh

Данный сервис является online компилятором с++

1) Открыть файл *.cpp 
2) Скопировать исходный код (Ctrl + A ) + (Ctrl + С)
3) Открыть http://cpp.sh вставить скопированный код в текстовое поле (Ctrl + A) 
4) Нажать кнопку Run.
5) Если нужно ввести данные с помощью клавиатуры в командной строке сайта.
